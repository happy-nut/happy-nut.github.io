# /diary - 개인 일기 생성

## 사용법
```
/diary [제목]
```

## 자동 카테고리 매핑 규칙

제목을 분석하여 카테고리 자동 추론:

| 키워드 | 카테고리 |
|--------|----------|
| 임신, 태동, 태명, 태몽, 출산 | `[육아, 출산]` |
| 아이, 육아, 키우다 | `[육아]` |
| 결혼, 혼인, 신고 | `[결혼]` |
| 이사, 집 | `[가족, 일상]` |
| 갈등, 화해 | `[결혼, 가족]` |
| 기본값 | `[육아, 출산, 결혼]` |

## 프로젝트 컨텍스트
- **경로**: `koongya/`
- **파일명**: `YYYY-MM-DD-제목.md`
- **기본 태그**: `[쿵야]` (자동 추가)

## Front Matter
```yaml
---
title: {제목}
date: YYYY-MM-DD HH:MM:SS +0900
categories: [{자동 추론된 카테고리}]
tags: [쿵야]
mermaid: true
math: false
---
```

## 실행 흐름
1. 제목 분석하여 카테고리 자동 추론
2. 오늘 날짜로 파일명 생성
3. 쿵야 태그 자동 추가
4. `koongya/` 디렉토리에 파일 생성

## 예시
```
/diary "첫 태동을 느끼다"
→ categories: [육아, 출산]

/diary "이사를 하다"
→ categories: [가족, 일상]

/diary "첫 갈등"
→ categories: [결혼, 가족]
```
