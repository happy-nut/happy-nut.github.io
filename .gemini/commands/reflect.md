# /reflect - 실수 회고

실수를 인식하고 바로잡은 뒤, **CLAUDE.md의 Lessons Learned 섹션에 교훈 기록** (재발 방지)

## 프로세스

1. **인식**: 사용자 피드백 경청, "~인데", "~아닌데" 표현 주목
2. **범위 파악**: 영향받은 파일 확인, 전체 복구 전 사용자 확인
3. **선택적 복구**: 잘못된 부분만 수정, 올바른 부분 유지
4. **원인 분석**: 어떤 가정/컨텍스트를 놓쳤는가?
5. **CLAUDE.md 업데이트**: Lessons Learned에 교훈 추가

## 교훈 기록 형식

```markdown
## Lessons Learned

- 교훈 내용 (간결하게)
```

또는 관련 섹션(Content Structure, Category 등)에 직접 규칙 추가

## 흔한 실수 패턴

| 유형 | 예시 |
|------|------|
| 컨텍스트 무시 | env 정보 미확인, 이전 맥락 놓침 |
| 과잉 수정 | 일부만 수정할 것을 전체 수정 |
| 가정 오류 | 검증 없이 추정, 패턴 잘못 일반화 |
| CLAUDE.md 미활용 | 기존 규칙 안 읽음, 교훈 미기록 |

## 출력

1. 잘못된 점 인정
2. 수정 범위 확인
3. 수정 실행
4. **CLAUDE.md Lessons Learned에 기록**
5. **(필수) 관련 커맨드 정의 파일도 보완** (예: /refine에 누락된 단계가 있으면 refine.md 수정)
6. (선택) 추가된 규칙 안내
